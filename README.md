# Marketing Mix Modeling con PyMC

![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)
![PyMC](https://img.shields.io/badge/PyMC-5.0+-green.svg)
![License](https://img.shields.io/badge/License-MIT-yellow.svg)

Proyecto académico: un pipeline modular para Marketing Mix Modeling (MMM) Bayesiano, preparado para análisis, visualización e interpretación con PyMC y Streamlit.

## Qué hay en este repositorio

- Código core (`mmm_core/`): transformaciones, construcción de modelo, métricas y visualización.
- Dashboard (`app/`): `app_mmm_streamlit.py` — interfaz interactiva para explorar resultados y descargar reportes.
- Datos de ejemplo (`data/`): `Basemediosfinal.csv`.
- Notebooks (`notebooks/`): EDA y ejemplos reproducibles.

## Inicio rápido

Sigue los pasos mínimos en `QUICKSTART.md` para instalar dependencias y ejecutar la app.

## Documentación y archivos focales

- `QUICKSTART.md`: guía mínima de instalación y ejecución (ideal para evaluadores y pruebas rápidas).
- `TROUBLESHOOTING.md`: soluciones a errores comunes (caché de Streamlit, errores de import, dependencias).
- `INSIGHTS_EXAMPLES.md`: ejemplos de los insights que el sistema genera (útil para reportes y presentaciones).
- `CHANGELOG.md`: historial de versiones y novedades (Fase 5, etc.).

Coloca problemas y preguntas en GitHub Issues si necesitas ayuda adicional.
